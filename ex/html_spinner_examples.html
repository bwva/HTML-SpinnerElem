Spun Out? Try the HTML Spinner Element

How is this spinner made?

These images illustrate how the cursor - the spinning part - is made from a square box with a border. One or more of the top, right, bottom, and left, borders are colored differently from the rest. Then the sides of the square are made round by giving the corners of the square a radius of 50% of its size. The result is four "quadrants" of the spinner to color and format for different spinners.

All of the spinners above use the same basic spinner with size 3em, cursor color green, and trace color light blue. The table shows how the spinner varies with different cursor types and weights, and the buttons change the spinner style for the whole set, identified by the class "demo".

Here is the spinner markup for one of the spinners above:
<div style="color:green;"> <x-spinner crsr="101" wt=".195" class="demo" tclr="rgba(0,55,255,.25)" style="font-size:3em;"></x-spinner> </div> yielding: , which still has the class "demo" so it will change cursor style when you click the buttons above.

Special Case: List Bullets

  • Customizable
  • No Dependencies or Frameworks
  • Compatible with standard HTML, css, and Javascript
  • Works in all modern browsers
  • May be entirely controlled in markup, or by script, or in combination
The spinning bullets on the list above are substituted for the list element's own bullet markers, by removing the list's own markers, and then inserting the custom spinner elements at the very start of each list item, just before its content. The insertion is accomplished with a brief script triggered after full DOM load to ensure the list is all there. See CodePen: Spinners as List Markers.
